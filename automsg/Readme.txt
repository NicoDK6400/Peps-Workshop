# Auto Message  

Dette er et script til dem der ønsker System beskeder.
Det kan bruges til at fortælle om discord, daglige genstart, keybind osv.

# Installation
Plug'n'play, kræver vRP!

Kig lige i client.lua INDEN du smider den på en Live server, der er noget der skal ændres inden, ellers bliver det sgu lidt pinligt for dig!!

# Inspiration
Ligger udgivet på FiveM forum og har taget nogle idéer derfra og lavet et simpelt script.
